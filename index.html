01

¿Que es Seguridad de redes?

La seguridad de red son aquellas acciones diseñadas y ejecutadas con el objetivo de salvo guardar el acceso, el uso y la integridad de la red y los datos contenidos en ella. Este concepto engloba aquellos elementos de hardware y software que conformen la red a la que se quiere implementar un sistema de seguridad.

02

¿Cómo funciona la seguridad de redes?

La seguridad de red es la combinación de diferentes capas de seguridad cada una con controles y modos de validación diferentes siendo esto para que los usuario realmente autorizados tengan acceso a los recursos de red y se bloquee o se dificulte lo las posible a los usuarios maliciosos para evitar que ataquen o se aorovechen de vulnerabilidades que no se tengan contempladas para implementar parches de seguridad que solventen esa vulnerabilidad.

03

¿Cuales son los tipos de seguridad de redes?

La seguridad de red son aquellas acciones diseñadas y ejecutadas con el objetivo de salvo guardar el acceso, el uso y la integridad de la red y los datos contenidos en ella. Este concepto engloba aquellos elementos de hardware y software que conformen la red a la que se quiere implementar un sistema de seguridad.

Firewalls

Los firewalls es una barrera entre una red interna y las redes externas, como Internet. Usan un conjunto de reglas específicas para permitir o bloquear el tráfico. Estos pueden ser ser hardware, software o ambos

Software antivirus y antimalware

El "malware" son virus, gusanos, troyanos, ransomware y spyware que infectan una red y permanecer el tiempo que haya programado el atacante. Los programas antimalware detectan la entrada de malware también hacen un seguimiento constante, eliminar malware y reparar daños.

Seguridad del correo electrónico

Los gateways para correo electrónicoes un factor de vulnerabilidad alta, haciendo uso de la información personal y las tácticas de ingeniería social Una capa de seguridad de correo electrónico bloquea los ataques entrantes.

VPN

Una VPN o red privada virtual es una red que encripta la conexión desde un terminal a la red por redes cómo internet haciendo uso de conexiones remotas y direcciones IP para autenticar las comunicaciones entre los dispositivos y la red.

04

¿Cuales son los tipos de ataques?

Una red siempre va a estar propensa a ataques estos pueden ser pasivos es decir cuando el atacante hace recepción de datos que viajan a través de la red y activos que es cuando el atacante hace uso de comandos para entorpecer el funcionamiento normal o cualquier otra acción maliciosa.

ataques pasivos

  1. Escucha telefónica
  2. Escáner de puertos
  3. Escaneo libre

ataques activos

  1. Ataque DDos
  2. Inyección SQL
  3. Phishing
  4. Desbordamiento de búfer
